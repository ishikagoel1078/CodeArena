# CodeArena

CodeArena is a full-stack online coding platform developed using HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB.

## Features

- User Registration & Login
- Secure Authentication
- Coding Challenges
- Problem Listings
- Leaderboard Ranking
- Submission History
- Responsive Dashboard
- MongoDB Database Integration

## Tech Stack

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas
